Hello, world!

Welcome to the car license plate detection demonstration.

This projects is for educational purposes to get into licences plate detection and working with Entity Framework.

We utilize the following compontents:

  • openalpr for license plate recognition.
  • PostreSQL as data storage.
  • pgAdmin to manage the data storage.
  • Apache Kafka for fast event drivate message distribution.
PostgreSQL, pgAdmin and Kafka are running in a dockerized environment.
